Open vs Closed Boundary

Open Source

  • PCG engine and APIs
  • collector contract
  • repository collector
  • local single-user dashboard
  • self-hosted runtime

Closed Source

  • org onboarding
  • OIDC and role mapping
  • scoped user/API tokens
  • audit logs and policy
  • admin workflows and enterprise UX
  • optional AI governance layer
