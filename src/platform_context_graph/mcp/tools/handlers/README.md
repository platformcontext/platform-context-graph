# MCP Tool Handlers

Each module in this package implements one family of MCP tool behavior:
- `analysis.py`: code search and relationship handlers.
- `ecosystem.py`: ecosystem and infrastructure handlers.
- `indexing.py`: indexing and package-discovery handlers.
- `management.py`: repository, bundle, and job handlers.
- `query.py`: read-only Cypher and visualization handlers.
- `watcher.py`: file-watcher lifecycle handlers.
