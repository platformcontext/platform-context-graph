# MCP Tools

This package defines the MCP tool manifest and the handler functions that back
those tools.

Structure:
- `codebase.py`: code indexing, search, and repository-management tools.
- `content.py`: portable content retrieval and content-search tools.
- `context.py`: entity and workload context tools.
- `ecosystem.py`: infrastructure, impact, and ecosystem tools.
- `handlers/`: callable implementations used by the MCP server.
